# Gmail Shortcut

| Key                | Description  |
| ------------------ | ------------ |
| <kbd>s-enter</kbd> | Send         |
| <kbd>a</kbd>       | Reply all    |
| <kbd>r</kbd>       | Reply        |
| <kbd>f</kbd>       | Forward      |
| <kbd>I</kbd>       | Mark as read |
| <kbd>s</kbd>       | Pin          |
| <kbd>!</kbd>       | Mark as spam |
| <kbd>e</kbd>       | Archive      |
| <kbd>v</kbd>       | Move         |
| <kbd>delete</kbd>  | Delete       |
